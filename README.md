# 🎯 Learning Dart

Repositório dedicado ao estudo e prática da linguagem **Dart**, criado para registrar o progresso de aprendizado desde os conceitos básicos até tópicos mais avançados.

---

## 📚 Sobre o Projeto

Este repositório serve como um diário de aprendizado da linguagem Dart. Aqui são armazenados exemplos de código, exercícios e anotações que cobrem os principais fundamentos e recursos da linguagem.

Dart é uma linguagem de programação desenvolvida pelo Google, focada em produtividade do desenvolvedor e usada principalmente com o framework **Flutter** para criação de aplicativos móveis, web e desktop.

---

## 🗂️ Tópicos Abordados

- Variáveis, tipos de dados e constantes
- Operadores e expressões
- Estruturas de controle (if/else, switch, loops)
- Funções e parâmetros (posicionais, nomeados, opcionais)
- Coleções: listas, conjuntos e mapas
- Programação orientada a objetos (classes, herança, interfaces, mixins)
- Null safety
- Programação assíncrona (async/await, Future, Stream)
- Generics
- Tratamento de erros e exceções

---

## 🛠️ Pré-requisitos

Antes de executar os exemplos, certifique-se de ter instalado:

- [Dart SDK](https://dart.dev/get-dart) (versão 3.0 ou superior)

Verifique a instalação com:

```bash
dart --version
```

---

## 🚀 Como Executar

Clone o repositório:

```bash
git clone https://github.com/tiagomatias930/Learnig-Dart.git
cd Learnig-Dart
```

Execute qualquer arquivo `.dart`:

```bash
dart run nome_do_arquivo.dart
```
